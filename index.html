Excuses

Excuses is a script that selects an excuse from a list at random and prints it out. Currently only available in python3.

What Can This be Used For?

Anything, such as:

  • Family
  • Work
  • Or any social event you would rather not take part in.

Usage

The script is written in python3. It's not backwards-compatable with python2.

From a terminal in the same directory as the script, run python3 excuses.py

The excuse is chosen and printed. Run it again or on a Linux system running bash, type !! as shorthand for the last command that was invoked on the system.
